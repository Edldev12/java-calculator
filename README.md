# Java Calculator

A simple console-based calculator built in Java that performs basic arithmetic operations using a menu-driven interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Input validation for division by zero
- Exit option

## Technologies Used

- Java
- Scanner Class
- Switch Statements
- Loops
